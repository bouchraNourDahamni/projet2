Tools

The drawing group contains pencil, brushes, pen, calligraphy and spraypaint.

Selection lets you select and transform your objects. Use left-click to select object, and right-click to invert a selection. Use mouse-wheel to rotate a selection, use the selection box to translate or rescale an object.

With eraser you can remove objects one at time with a single click, or everything the eraser touches by holding left-click.

Shapes allows you to create ellipses, a rectangles and polygons.

Line lets you create straight segments.

Colors provides a filling color bucket or a color application option which allows you to change the color of an object or its outline.

With Stamp you can choose to add a stamp from our limited selection. Use mouse-wheel to rotate a stamp before adding it.

You can toggle the grid with "g", and set its size.
